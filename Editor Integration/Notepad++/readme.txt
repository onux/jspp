Notepad++ Plugin for JS++
---
This is a Notepad++ plugin for JS++ to support syntax highlighting, code folding, etc. This plugin was developed in and exported from Notepad++ 6.9.2.

Installation
-
1. In the "Language" menu, select "Define your language..."
2. The "User Defined Language" dialog should appear. Click the "Import..." button.
3. Import the .xml file
4. After the import is successful, you will be brought back to the "User Defined Language" dialog. At the top left, there should be a drop-down menu. Select "JS++" from the drop-down menu.
5. Close the dialog and restart Notepad++.
6. Open a JS++ file. In the "Language" menu, select "JS++".
7. Done! All JS++ features should now be supported in Notepad++.

The official documentation for Notepad++ also provides instructions for installing languages:

http://docs.notepad-plus-plus.org/index.php/User_Defined_Language_Files#How_to_install_user_defined_language_files
